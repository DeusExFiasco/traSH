# traSH
## A tiny bash clone
###### With full environment, process piping, stream redirection and batch execution support.

### About

This is a personal project I started as a YouTube series on how to write a simply bash-like shell in C with only standard libraries.
It is heavily based on the School42 project "minishell", which I consider to be one of the most enlighting learning experiences in my
programming career. I wanted to share the knowledge that writting a basic shell like this involves with a wider audience who shares
my curiosity for how every-day systems we often take for granted actually work. The name is just a silly joke I thought of while reading
the original bash source, **which is orders of magnitude more complex than this!**
